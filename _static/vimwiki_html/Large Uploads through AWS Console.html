Managing Large Uploads with the AWS console

When uploading large datasets to NeuroCAAS, you may experience difficulties through the native website interface. In particular, some users have experienced difficulties with datasets greater than 2 GB. To provide a fast and easy solution to this issue, we provide an alternative upload option with optimized upload times through the Amazon Web Services (AWS) console.

Step 1: Getting Credentials

To get started, you will need:

  1. the data you want to upload
  2. AWS Credentials.

You can get AWS credentials in your account profile (click your name on the website):

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep0.png

Then note your AWS User Name and AWS Password. You will be prompted for these credentials later on.

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep1.png

Step 2: Logging in to AWS

Next, we will login to AWS. You can access AWS through the link at the bottom of each analysis page:

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep2.png

This will take you to the AWS login page. On this first page, make sure to click "IAM user" instead of "Root user", and enter "neurocaas" in the Account ID box:

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep3.png

Finally, you will be asked to enter an IAM user name and Password for the NeuroCAAS account. Enter the AWS User Name and AWS Password here from your account profile:

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep4.png

Step 3: Uploading Data

Now, you should see a page that says "Amazon S3". You will be dropped into a folder that represents your personal data storage for the particular analysis page that you logged in through. Note that you can also freely upload, organize and delete data here, but you will not be able to download input data back to yourself, access to any other user data, or any other features of AWS. Go ahead and drag and drop the data you want to upload onto the page:

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep5.png

You will then be taken to a summary page where you will be asked to confirm the upload that you started (total number of files, file size, etc.). If everything looks good, click the orange Upload button:

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep6.png

The file will then begin uploading. Note that you should not interrupt the internet connection (putting your computer to sleep, for example) while this is going on, as it will cause the upload to fail.

file:~/neurocaas/media/awsloginwalkthrough/awsloginstep7.png

Once the upload has succeeded, you will see the new data in the page. Once you go back to the NeuroCAAS page, you will see your data there as well, ready to be analyzed in jobs.
