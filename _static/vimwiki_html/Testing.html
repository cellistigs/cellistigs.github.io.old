Github Actions

Github actions is now my standard for CI. It does not have the same run limitations as Travis, and has great integration and formatting tools.

Docker

To integrate github actions with Docker, use the docker login action, and make sure your repo is set up with a docker username and access token: https://docs.docker.com/docker-hub/access-tokens/
