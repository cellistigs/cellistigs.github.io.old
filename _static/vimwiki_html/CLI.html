Documentation written for the process of using the cli while we test it.

  1. install the neurocaas-contrib repository locally.
  2. activate an AWS instance with it.
  3. log in.
  4. install the neurocaas-contrib repository remotely.
  5. run neurocaas-contrib workflow initialize-job -p contribdata
    1. insight: if the videoname already exists at the right location, you don't have to worry about all your functions complaining when you run get-data.
