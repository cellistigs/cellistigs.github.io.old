Chases and Escapes

This section is devoted to the challenge problems in the book Chases and Escapes by Paul J. Nahin.

Chapter 1: The Classic Pursuit problem

  1. What is the total distance traveled by the pirate vessel before capture for n < 1? We can answer this question by calculating the time, t, that is taken for capture to occur. This can be done as follows.
    • Calculate the y position at which capture occurs.
    • Calculate time from the y position via the merchant vessel's velocity, as \(t = \frac{y}{V_m}\)
    • Calculate the distance traveled by the pirate vessel d as \(d = V_d*t\)

Thus for a given \(x_0, V_m, V_p\),

\begin{align} d &= \frac{V_d}{V_m}*\frac{n}{1-n^2}x_0 \tag{$n = \frac{V_m}{V_p}$} \\ d &= \frac{x_0}{1-n^2} \end{align}
  1. The tail chase lag distance is the distance at which the pirate ship trails the merchant ship in the case where \(n = 1\), or \(V_m = V_p\). Critically, this means that the two vessels cover distance at the same rate, and that at any time they have moved the same distance from their original starting points. We can use this to calculate the distance traveled by the merchant ship as a function of x, using the formula for the total distance traveled as a function of time.
