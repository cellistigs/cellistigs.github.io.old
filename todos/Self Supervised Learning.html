Self Supervised Learning

The defining feature of self supervised learning is that you're going to learn a good representation of the data by looking at some aspect of your own training.

InfoNCE

In the case of Oord et al. 2019 (Contrastive Predictive Coding) you do this by means of negative samples in the context of sequential data: given some sequence data, we try to learn a good representation of the data by:

  • 1. Forming a per-timepoint (or more coarse) representation of the data using an encoder model (\(z_t\))
  • 2. Transforming the per-timepoint representation with an autoregressive model (\(c_t\))
  • 3. Estimating the unnormalized likelihood of future sequence data based on a bilinear form (\(c_t^TW_{t+k}z_{t+k}\))
  • 4. Optimizing this estimate. This is essentially a classification task that asks networks to correctly identify the data at a certain point in the future given a latent representation of the present and latent representations of the future.
  • 5. By doing so, one can learn latent representations of the data and a long term context that capture some information about the future of this data.
